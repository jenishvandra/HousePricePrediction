# House Price Prediction

## XYlofy AI — Internship Project

## Project Overview
Built a regression model to predict house prices based on property features like area, bathrooms, airconditioning, and location using the Kaggle Housing Prices Dataset (545 records, 13 features).

## Models Used
- Linear Regression — Test R² : 0.6529
- Random Forest Regressor — Test R² : 0.6124
- Winner: Linear Regression

## Files
- `analysis.ipynb` — Complete Jupyter Notebook (all 5 tasks)
- `Housing.csv` — Dataset (Kaggle)
- `summary.pdf` — Project findings report
- `charts/` — All visualization charts
